# Cole Snipes Portfolio

Welcome to my personal developer portfolio!  
This site highlights my projects, skills, and experience as a Computer Science student at the University of Indianapolis, I just recently graduated from University on May 2026. It’s designed to be clean and responsive.


---

## Features
- Single-page portfolio (Hero, Work, About, Contact) with editorial design system  
- React + Vite build with lightweight CSS and semantic HTML  
- Serverless contact form (Resend) with rate limiting and spam protection  
- Responsive layout with scroll-reveal transitions  


---

## Tech Stack
- **Frontend:** React 18 + Vite  
- **Styling:** CSS custom properties, editorial grid, responsive layout  
- **Serverless:** Vercel Functions + Resend email API  
- **Deployment:** Vercel (recommended)  


---

## Projects Highlighted
Projects live in `src/data/projects.json` and are rendered in the Work section.
